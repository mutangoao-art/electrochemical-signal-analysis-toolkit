# Electrochemical Signal Analysis Toolkit

A Python toolkit for preprocessing, visualization and feature extraction of electrochemical sensing data.

## Planned Features

- Import electrochemical data
- CV visualization
- Baseline correction
- Signal smoothing
- Peak detection
- Feature extraction
- EIS analysis
- Automatic report generation
